# SALAMANDER · 橫版飛行射擊遊戲

一款用純 HTML5 Canvas + JavaScript 製作的橫版飛行射擊小遊戲，靈感來自經典街機《沙羅曼蛇 (Salamander / Life Force)》。零依賴、單一檔案，用瀏覽器打開 `index.html` 即可遊玩。

## 🎮 線上試玩

開啟 GitHub Pages 後，即可透過以下連結直接遊玩：

```
https://<你的使用者名稱>.github.io/<倉庫名稱>/
```

（設定方式見下方「部署到 GitHub Pages」）

## 操作方式

| 按鍵 | 功能 |
|------|------|
| 方向鍵 / WASD | 移動戰機 |
| 空格 / Z | 持續射擊 |
| 觸控（手機） | 拖曳移動、按住螢幕射擊 |

## 道具系統

擊敗敵機有機率掉落道具：

| 圖示 | 名稱 | 效果 |
|------|------|------|
| L | 雷射炮 | 子彈變為高速雷射，持續一段時間 |
| M | 多重彈 | 升級為 2 發 → 3 發散彈 |
| S | 護盾 | 抵擋一次傷害 |
| ♥ | 生命 | 增加一條命 |

## 敵機類型

- **紅色戰鬥機** — 快速衝刺，不會射擊
- **灰色坦克機** — 耐打，需多發才能擊毀
- **紫色炮手** — 會朝玩家追蹤射擊
- **大型 Boss** — 每隔一段時間刷新，有血條，發射散彈彈幕

遊戲會隨時間自動提升等級（LV），敵機數量與速度逐漸增加。

## 🎵 音效與音樂

遊戲內建純程式合成的 8-bit 復古音效與背景音樂，**完全使用 Web Audio API 即時生成，不含任何外部音訊檔案，無任何版權問題**：

- **背景音樂** — Am → F → C → G 循環琶音，方波主旋律 + 三角波貝斯 + 噪音 hi-hat
- **音效** — 射擊、爆炸、受擊、護盾破裂、拾取道具、Boss 出現
- 介面右側有 **🔊 音效開 / 🔇 音效關** 切換按鈕

> 註：因瀏覽器的自動播放政策，音訊會在你點擊「開始遊戲」後才啟動。

## 本地運行

不需要任何建置工具，直接用瀏覽器打開即可：

```bash
# 方式一：雙擊 index.html

# 方式二：用本地伺服器（推薦，避免部分瀏覽器限制）
python3 -m http.server 8000
# 然後瀏覽器打開 http://localhost:8000
```

## 部署到 GitHub Pages

1. 將本倉庫推送到 GitHub
2. 進入倉庫的 **Settings → Pages**
3. **Source** 選擇 `Deploy from a branch`，分支選 `main`、目錄選 `/ (root)`
4. 儲存後等待約 1 分鐘，即可透過 `https://<使用者名稱>.github.io/<倉庫名稱>/` 遊玩

## 技術說明

- 純前端，無任何外部依賴或框架
- HTML5 Canvas 2D 繪圖
- 包含星空背景、粒子爆炸、碰撞偵測、難度遞增等系統
- 支援桌面鍵盤與行動裝置觸控
